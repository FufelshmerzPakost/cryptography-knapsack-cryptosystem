# Рюкзачное шифрование (На основе задачи об укладке рюкзака)

Входные данные:
- Сообщение (Не более 19 символов, ограничение u32, надо поковырять под свой конфиг)
- N любое число
- M любое число (Больше максимального в публичном ключе)
